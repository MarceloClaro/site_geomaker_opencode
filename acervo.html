Como instalar, criar a coleção e conectar ao site
  1. 01
    Prepare um WordPress

    Use um domínio ou subdomínio institucional, por exemplo acervo.seudominio.org.

  2. 02
    Instale o plugin Tainacan

    No WordPress, abra Plugins → Adicionar novo, procure “Tainacan”, instale e ative.

  3. 03
    Crie a coleção

    No menu Tainacan, escolha Nova coleção, defina os metadados e adicione as peças com suas imagens.

  4. 04
    Copie o ID da coleção

    Abra a coleção no painel. O número exibido na URL ou retornado pela API é o ID usado pelo site.

  5. 05
    Preencha a configuração

    Edite assets/config.js com a URL pública, o ID e o endereço administrativo.

assets/config.js

tainacanBaseUrl: "https://acervo.seudominio.org",
tainacanCollectionId: "12",
tainacanAdminUrl: "https://acervo.seudominio.org/wp-admin/",

Depois de salvar e publicar novamente o site, o botão acima abrirá o painel e o catálogo passará a buscar os registros públicos.

Documentação

Metadados com propósito educativo.

Cada registro poderá combinar identificação museológica, contexto territorial, direitos de uso e uma camada de mediação para professores e estudantes.

O modelo é compatível com taxonomias, filtros e mapeamento para padrões como Dublin Core. Imagens e textos deverão ser publicados apenas com origem e licença verificadas.
