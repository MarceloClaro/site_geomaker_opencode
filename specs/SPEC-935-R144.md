# SPEC-935-R144 — Console Error Analyzer (classificação automática de logs do navegador)

## Objetivo

Criar um analisador automático de logs de console do navegador que
classifique cada linha em categorias (extensão Chrome, infraestrutura,
informativo, erro real de código) e produza um relatório estruturado,
substituindo a análise manual do `erro.txt`.

## Motivação

O `erro.txt` do Geomaker era uma coleção manual de ~155 linhas de console
do navegador, misturando erros reais de extensões Chrome com avisos de
infraestrutura e navegação. A correção manual é frágil e não reproduzível.

Um analisador programático:
- Torna a análise reproduzível (mesma entrada → mesma saída)
- Classifica automaticamente por padrões conhecidos
- Gera relatório estruturado (JSON + texto)
- Pode ser integrado a CI/testes

## Escopo

### Em escopo
- `scripts/console_error_analyzer.py`: módulo com:
  - `classify_line(line: str) -> dict` — classifica uma linha do console
  - `analyze(lines: list[str]) -> dict` — processa múltiplas linhas
  - `generate_report(analysis: dict) -> str` — gera relatório legível
  - Padrões de classificação:
    - `extension`: extensões Chrome (eesel AI, searchitfastnow, refresh,
      single-player, agent-chat)
    - `infrastructure`: Cloudflare 524, WebSocket localhost
    - `informational`: THREE.WebGLRenderer, navegação, lazy load,
      background ready
    - `unknown`: linha não reconhecida
- `tests/test_r144_console_error_analyzer.py`: TDD dos classificadores
- Aplicação ao `erro.txt` atual

### Fora de escopo
- Parse de stack traces completas (linhas de continuação `at ...`)
- Suporte a todos os navegadores (apenas Chrome)
- Integração com sistema de logging em tempo real

## Critérios de Aceitação

1. `classify_line("content.bundle.js:1 Initialized eesel AI")` →
   `{"category": "extension", "source": "eesel AI", "severity": "info"}`
2. `classify_line("main:1 ... status of 524 ()")` →
   `{"category": "infrastructure", "source": "cloudflare", "severity": "warning"}`
3. `classify_line("THREE.WebGLRenderer 73")` →
   `{"category": "informational", "source": "three.js", "severity": "info"}`
4. `classify_line(...)` com extensão `biaggnjib...` →
   `{"category": "extension", "source": "searchitfastnow"}`
5. `classify_line(...)` com `refresh.js:27 WebSocket` →
   `{"category": "extension", "source": "refresh"}`
6. `classify_line("single-player.bundle.js ... agent-chat")` →
   `{"category": "extension", "source": "single-player"}`
7. `classify_line("Navegou para ...")` →
   `{"category": "informational", "source": "navigation"}`
8. `analyze([])` → `{"total": 0, "categories": {}, "lines": []}`
9. `generate_report(analysis)` contém "ZERO ERROS DE CÓDIGO"
10. `python3 scripts/console_error_analyzer.py /caminho/erro.txt`
    produz saída com categorias e resumo.
